User types

  • Super admin: Can access everything (all accounts, create/edit/remove accounts, manage all sections and items, manage all users with customisation settings, access all settings).
  • Admin: Can access allocated accounts, create/edit/remove accounts, sections, and items, create admin and user logins for accounts they manage, and user customisations inherit the creator’s settings. Access limited settings.
  • Standard user: Access allocated accounts, sections, and items only; no access to settings.

Customised label settings for users are only visible to super admins.
